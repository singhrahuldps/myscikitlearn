# myscikitlearn
My implementation of some Machine Learning Algorithms from scratch.

## Required libraries -
* Numpy
* Pandas
* Sklearn for accuracy_score

## Basic Usage
```
clf = Classifier()
clf.fit(x_train, y_train)
pred = clf.predict(x_test)
```
